## Setup — Step by Step

### Step 1 — Get the Crop Recommendation Dataset

1. Go to: https://www.kaggle.com/datasets/atharvaingle/crop-recommendation-dataset
2. Download `Crop_recommendation.csv`
3. Place it in the `notebooks/` folder
4. This is a REAL dataset — your accuracy will jump from 16% to 99%

### Step 2 — Train the model

```bash
cd notebooks
pip install scikit-learn pandas numpy
python train_model.py
# This generates: backend/ml/model.pkl, scaler.pkl, label_encoder.pkl, metadata.json
```

Expected output:
```
Test Accuracy: 0.9909 (99.1%)
CV Mean Accuracy: 0.9887 ± 0.0024
```

### Step 3 — Set up the backend

```bash
cd backend
pip install -r requirements.txt
```

Create a `.env` file in backend/:
```
JWT_SECRET=your-secret-key-change-this
OPENWEATHER_KEY=your-key-from-openweathermap.org   # optional
DATABASE_URL=sqlite:///./agrisetu.db               # or PostgreSQL URL
```

Start the backend:
```bash
uvicorn main:app --reload
# API running at http://localhost:8000
# Docs at http://localhost:8000/docs
```

### Step 4 — Set up the frontend

```bash
cd frontend
npm install
```

Create `frontend/.env`:
```
REACT_APP_API_URL=http://localhost:8000
```

Start the frontend:
```bash
npm start
# App running at http://localhost:3000
```

## API Endpoints

### Auth
| Method | Endpoint | Description |
|---|---|---|
| POST | /auth/register | Register farmer or buyer |
| POST | /auth/login | Login, returns JWT token |

### Crop Advisory
| Method | Endpoint | Description |
|---|---|---|
| POST | /crops/predict | Submit soil data → get top 3 crops |
| GET | /crops/history/{user_id} | Past recommendations |
| GET | /crops/crops-list | All 22 crops the model supports |

### Marketplace
| Method | Endpoint | Description |
|---|---|---|
| GET | /marketplace/products | Browse all listings |
| POST | /marketplace/products | Farmer creates listing |
| PUT | /marketplace/products/{id} | Update price/quantity |
| POST | /marketplace/orders | Place an order |
| PATCH | /marketplace/orders/{id}/status | Update order status |
| GET | /marketplace/stats/farmer/{id} | Revenue + order stats |

### External
| Method | Endpoint | Description |
|---|---|---|
| GET | /external/weather?city=Kolkata | Current weather |
| GET | /external/mandi-prices?crop=Rice | Current mandi price |

## Free Hosting

### Backend on Render.com
1. Push to GitHub
2. New Web Service → connect repo
3. Set root directory to `backend`
4. Build command: `pip install -r requirements.txt`
5. Start command: `uvicorn main:app --host 0.0.0.0 --port $PORT`
6. Add environment variables

### Frontend on Vercel
1. New Project → import from GitHub
2. Set root directory to `frontend`
3. Add env: `REACT_APP_API_URL=https://your-render-url.onrender.com`
4. Deploy
